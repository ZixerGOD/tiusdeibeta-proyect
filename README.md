<h2>This app is UI/UX pixel perfect clone of 'monday.com'.</h2>

<img src='/readme-media/readme-png/01.png' /> 

Check it out here: https://anyday-za1z.onrender.com/

## Project Features

- Templates - the user can select a template board.

- Custom - the user can manually create his board, give it a name, add groups and tasks, change colors, set time estimation, priority and more.

<img src='/readme-media/readme-gif/01.gif' /> 

- Login \ Signup, the user can signup with an email, full name and a password after signing in he can upload a profile picture.

- There is also the option to signin via facebook (using FB API to signin new users).

- We also support 'guest mode', that means the user can fully work on a board as a guest.

<img src='/readme-media/readme-png/02.png' width="600" /> 
<img src='/readme-media/readme-png/03.png' width="600" /> 

- We support full crudl (Create | Read | Update | Delete | List).

- The user can create a new board \ group \ task \ comment.

- The user can update the title \ label \ priority \ time-line \ add or remove team members.

<img src='/readme-media/readme-gif/02.gif' width="600" /> 
<img src='/readme-media/readme-gif/03.gif' width="600" /> 

- The user can add\delete comments to each task and pin\un-pin a comment.

- The user also can upload an image \ video url \ edit text and more...

- For each specific task there is an activity log which contains all the changes done to the task.

<img src='/readme-media/readme-gif/04.gif' width="600" /> 

- The user can multi select tasks\groups with check-box.

- The selected tasks\groups can be deleted, duplicated, or moved in to a another group.

<img src='/readme-media/readme-gif/05.gif' width="600" /> 

- The user can view his board as kanban.

- There is a dashboard containing all boards including number of boards/number of tasks in each state and more.

<img src='/readme-media/readme-gif/06.gif' width="600" /> 
<img src='/readme-media/readme-gif/07.gif' width="600" /> 

- The user can DND tasks\groups & lables vertically and horizontally.

- When draging a group, all groups will automatically collapse. When the drag is finished they will automatically uncollapse .

- DND is also active on kanban view.

<img src='/readme-media/readme-gif/08.gif' width="600" /> 

- The user can do things with voice controll like adding a group or task | filter the board | close voice manger.

<img src='/readme-media/readme-gif/09.gif' width="600" /> 

- The users can see every change live (the app uses sockets to emit all users on every change)

- Our app is responsive to each device, desktop \ tablet \ mobile

<img src='/readme-media/readme-gif/10.gif' width="600" /> 
<img src='/readme-media/readme-gif/11.gif' width="600" />

## Installation
To run the project locally, follow these steps:

Clone the repository: 
`git clone https://github.com/yosikari/Anyday.git`

Open two terminals `cd frontend` | `cd backend`

On each terminal run the following commands,

Install dependencies: 
`npm install`

Start the server: 
`npm start`

**The website should be available at** http://localhost:3000/.
